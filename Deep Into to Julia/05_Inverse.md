## Lines

Tangent Lines

````julia
f(x) = sqrt(x)
c = 2
tl(x) = f(c) + 1/(2 + sqrt(2)) * (x - c)
xs = range(0, 3, length = 150)
ys = f.(xs)
zs = tl.(xs)
plot(xs, ys, color = :blue, legend=false)
plot!(xs, zs, color = :blue)
plot!(ys, xs, color = :red)
plot!(zs, xs, color = :red)
````
